## Proboszcz

My current Hackintosh PC, bought as post-lease from FIAT in 2022.  
Installed macOS Sonoma on it with my own OpenCore EFI made following the [OpenCore Install Guide by Dortania](https://dortania.github.io/OpenCore-Install-Guide/).

### Specifications

- Processor: Intel Core i5-8250U @ 1.60 GHz
- Model: HP ProBook 650 G5
- Memory (Total: 16GB):
    - Slot 0: Samsung 16GB DDR4 2400MT/s
    - Slot 1: Empty
- Mass storage:
    - Boot drive: Lexar NM520 512GB NVMe SSD
    - Testing drive: WD Green 120GB SATA SSD
- Graphics card: Intel UHD Graphics 630 (Integrated)
- Display: 1920x1080 IPS LCD 15" 60Hz

### Accessories:

- Power supply: HP 45W

### Uses:

- Web development
- Testing
